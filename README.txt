== Description ==
LM Easy Maps is a plugin designed to make easy to insert maps into a post, page or widget.

The installation is very simple, just upload the .zip file through the WordPress plugin manager, or look for it in the plugin directory and install it with the automatic procedure.

The plugin uses maps of Google, it is possible, on the admin side, to use a point of interest as start point on the map by entering an address, gps coordinates or a point of interest, it is also possible to decide the display size of the map both with fixed values in pixels, and with variable values expressed as a percentage.

== Installation ==
 
This section describes how to install the plugin and get it working.
 
1. Upload 'lm-easy-maps' directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
